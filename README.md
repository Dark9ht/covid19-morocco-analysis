# COVID-19 Analysis in Morocco

## Project Overview
This project analyzes COVID-19 data in Morocco to understand trends over time, regional distribution, and case progression.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Questions
- How did COVID-19 cases evolve over time?
- Which regions were most affected?
- How do confirmed, recovered, and death cases compare?

## Key Findings
- COVID-19 cases increased significantly over time.
- Casablanca-Settat recorded the highest number of cases.
- Recovered cases closely followed confirmed cases.
- Death counts remained relatively low compared to total cases.

## Conclusion
The analysis highlights the evolution of COVID-19 across regions and over time in Morocco, showing both temporal trends and regional disparities.
